# AI-for-Med-week07
# AI Medical Assistant Website

## ข้อมูลผู้จัดทำ
- **ชื่อผู้จัดทำ:** นายณัฐภัทร ปัทมะทิน
- **โรงเรียน:** เตรียมอุดมศึกษา
- **สาขาที่สนใจเข้า:** คณะวิศวกรรมศาสตรบัณฑิต สาขาวิชาวิศวกรรมคอมพิวเตอร์ จุฬาลงกรณ์มหาวิทยาลัย

## เกี่ยวกับโปรเจ็กต์
AI Medical Assistant เป็นเว็บไซต์ที่พัฒนาขึ้นเพื่อช่วยให้ผู้ใช้งานสามารถรับคำแนะนำทางการแพทย์เบื้องต้นได้อย่างสะดวกและปลอดภัย โดยเว็บไซต์นี้สามารถ:  
1. จัดเก็บและเข้าถึงฐานข้อมูลของผู้ป่วยอย่างเป็นระบบ  
2. วิเคราะห์ภาพทางการแพทย์ เช่น X-ray ปอด และภาพเซลล์เม็ดเลือดขาว ด้วยเทคโนโลยี AI  
3. ให้คำแนะนำทางสุขภาพเบื้องต้นตามผลวิเคราะห์  

โปรเจ็กต์นี้ถูกออกแบบให้ใช้งานง่าย และสามารถใช้เป็นเครื่องมือช่วยสนับสนุนการวินิจฉัยเบื้องต้นของผู้ป่วยหรือบุคคลทั่วไป

## ฟีเจอร์หลัก
- ระบบจัดการข้อมูลผู้ป่วย  
- การวิเคราะห์ภาพทางการแพทย์ด้วย AI  
- การให้คำแนะนำทางการแพทย์เบื้องต้น  
- อินเทอร์เฟซใช้งานง่าย รองรับผู้ใช้ทุกกลุ่ม  

## เทคโนโลยีที่ใช้
- Frontend: HTML, CSS, JavaScript  
- Backend: (ระบุภาษา/Framework ที่ใช้ เช่น Python Flask, Node.js)  
- AI/ML: (ระบุโมเดลหรือไลบรารี เช่น TensorFlow, PyTorch, OpenCV)  
- Database: (ระบุชนิด เช่น MySQL, MongoDB)
